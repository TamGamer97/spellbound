# Data folder

**Canonical files (used by the app and scripts):**

| File | Description |
|------|--------------|
| `puzzles-2.json` | **Main puzzle set** — used by the game (lobby, versus, solo). Generated by `scripts/v2/index.js`. |
| `bad-pangrams.txt` | Words to exclude as pangrams (one per line). Loaded by `scripts/cleanup-puzzles-2-pangrams.js`. Source: words marked ✗ in `pangram-review.txt`. |
| `pangram-review.txt` | Human review of all pangrams: ✓ = keep (common English), ✗ = remove. Used to maintain `bad-pangrams.txt`. |
| `common-7-letter-words.txt` | Common 7-letter word list (e.g. poslarchive). Used as pangram candidates by the v2 generator. |
| `wiki-100k.txt` | Word list for generator allowed set and enrichment. |

**Archive (`data/archive/`):**

Legacy or experimental files not used by the app:

- `puzzles-v1-legacy.json` — Output of v1 generator (old 15-puzzle set).
- `puzzles-v3-experimental.json` — Output of v3 Python generator (optional).
- `pangram-words-extract.txt` — One-time extraction of unique pangrams (redundant with `pangram-review.txt`).
- `pangram-candidates.txt` — Small manual candidate list (reference).
